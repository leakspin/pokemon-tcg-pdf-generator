Here I will explain what it everything really does:

1. Regulation: selects the regulation that you want to fill. As of now, only Scarlet & Violet is available, but future regulations could be added

2. Download PDF: it generates the final PDF and downloads it to your system.

3. Save data: this button generates a JSON file that contains all the info from your deck. If you save this, you can later load it to recover the data in this same page and modify the data, if you want, or download the PDF again.

4. Load data: this button prompts a dialog where you need to select a JSON file compatible with this tool. Then, it will load it, filling all the data.
